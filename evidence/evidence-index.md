# Evidence Index

## Purpose

The evidence index tracks where supporting artifacts are stored for each case study.

## Evidence Types

- dashboards
- reports
- interview notes
- architecture diagrams
- policy or review documents
- before / after metrics

## Index Fields

- case name
- evidence type
- source
- date
- owner
- notes

## Use

Use this index to keep supporting artifacts easy to find, review, and reuse when a case study is updated.
